纯 C++ 实现的 Windows 平台轻量工具，绕过软件粘贴 API 检测 / 剪贴板屏蔽，通过模拟原生键盘击键实现文本粘贴，适配各类禁用粘贴的窗口场景。

简介

针对 Windows 平台下部分软件（如考试系统、客户端软件、网页限制）通过Hook 剪贴板 API、拦截粘贴消息、检测粘贴操作等方式禁用粘贴的问题，本工具不使用系统标准粘贴 API，而是将剪贴板文本转换为键盘输入指令，模拟人工逐键敲击，完美绕过各类粘贴检测机制。


纯 Windows API 实现，无第三方依赖
轻量极简，编译后单文件运行
支持中英文、数字、特殊字符全类型输入
兼容 Windows 10/11 32/64 位系统


核心功能


绕过粘贴检测
绕过 WM_PASTE 粘贴消息、GetClipboardData 剪贴板 API Hook、粘贴行为检测等限制，不触发任何粘贴告警。


原生键盘模拟
基于 Windows 官方 SendInput API 实现键盘模拟，还原真实击键逻辑，与人工输入无差异。


一键自动粘贴
读取系统剪贴板文本 → 自动模拟键盘输入 → 输出到当前焦点窗口，全程无需手动操作。


全字符支持
支持 ASCII 字符、中文、符号、空格、换行等所有常规输入文本。


实现原理
常规粘贴：软件调用剪贴板 API / 发送粘贴消息，易被检测拦截；
本工具：完全弃用粘贴相关 API，读取剪贴板文本后，将字符转换为 Windows 虚拟键码 unicode，通过SendInput模拟键盘按下 + 释放动作，实现「人工打字」效果，从根源绕过粘贴检测。


快速使用
运行工具：双击 exe 文件；
复制文本：正常复制需要粘贴的内容（Ctrl+C）；
聚焦窗口：打开目标限制软件，将光标定位到需要输入的输入框（确保窗口处于激活焦点）；
开始粘贴：使用ctrl + shift + v快捷键；
完成输入：等待文本自动输入完毕即可。
